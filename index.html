Getting Started

Quick setup guide to start using Speak2Text effectively

Microphone & Permissions

To start using Speak2Text:

  • Open Settings > Privacy > Microphone
  • Enable Speak2Text access
  • Allow Speech Recognition when prompted
  • Grant Calendar access for date detection

Language Configuration

Set up your preferred languages:

  • Tap the language icon in the top toolbar
  • Select source language for speech recognition
  • Choose target languages for translation
  • Save your preferences

Translation Setup

For real-time translation:

  • App handles API keys automatically
  • No manual setup required
  • Premium unlocks all 20+ languages
  • Free version includes 5 languages

Troubleshooting Guide

Solutions to common issues

Microphone Not Working

  • Check iOS Settings > Privacy > Microphone - enable Speak2Text
  • Ensure microphone is not blocked or covered
  • Restart the app and your device
  • Check for iOS updates

Speech Recognition Issues

  • Ensure you have a stable internet connection
  • Check that correct language is selected
  • Reduce background noise
  • Speak clearly and at natural pace

Translation Not Working

  • Verify internet connection
  • Check target language selection
  • Restart the app
  • Ensure you're not in airplane mode

App Freezes or Crashes

  • Update to the latest version of Speak2Text
  • Restart your device
  • Clear app cache (Settings > Speak2Text > Clear Cache)
  • Reinstall the app if issues persist

Battery Optimization

  • Enable Low Power Mode during extended use
  • Reduce screen brightness
  • Close background apps
  • Update to latest iOS version

Frequently Asked Questions

Find quick answers to common questions

Which languages does Speak2Text support?

Speak2Text supports 12+ source languages for speech recognition and 20+ languages for translation, including:

  • English, Spanish, French, German, Italian
  • Chinese, Japanese, Korean, Arabic, Russian
  • Portuguese, Dutch, Swedish, Turkish, Hindi
Does Speak2Text work offline?

Yes, the speech-to-text functionality works offline for enhanced privacy. However, translation features require an internet connection as they utilize Microsoft Azure's cloud-based translation services.

How is my voice data handled?

Privacy is our top priority:

  • Voice processing happens entirely on your device
  • No voice recordings are stored or sent to servers
  • Transcripts are only saved if you choose to save them
  • We don't sell or share your data with third parties
  • All data is encrypted and GDPR compliant
How can I improve transcription accuracy?

For best results:

  • Use in a quiet environment
  • Position your device 6-12 inches from your mouth
  • Speak clearly at a natural pace
  • Ensure you have the correct language selected
  • Update to the latest version of the app
  • Use premium for advanced noise cancellation
How do I export my notes?

Exporting notes is simple:

  • Open the note you want to export
  • Tap the share icon in the top right corner
  • Choose your export format (Text, PDF)
  • Select where to send/save the file
  • Premium users get additional export options
Why do I need an API key for translation?

Speak2Text automatically handles API keys for Microsoft Azure Translator services:

  • No manual setup is required for most users
  • The app manages keys securely in the background
  • Your key is stored securely in iOS Keychain
  • We never access or store your key on our servers
How does the AI content analysis work?

Our AI analyzes your content to provide:

  • Key insights and summaries of long content
  • Automatic extraction of action items
  • Sentiment analysis of conversations
  • Content improvement suggestions
  • Advanced analysis for premium users
Is Speak2Text accessible for users with disabilities?

Yes, we prioritize accessibility:

  • Full VoiceOver support
  • High contrast mode
  • Adjustable text sizes
  • Compatibility with Switch Control
  • Designed following WCAG guidelines
